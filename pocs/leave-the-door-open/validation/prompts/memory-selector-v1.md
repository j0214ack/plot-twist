# Eligible Memory Selector v1

Choose background context for one character's next private-self-talk turn.

You receive only memory cards already made eligible by the Controller. You
cannot ask for other memories. The player's thought is a topic cue, not proof
that its claims are true. Select no card unless one card is directly useful to
answer the player's current thought or preserve continuity in the present
situation. A direct question about an eligible ordinary-biography fact is
sufficient relevance. Otherwise ordinary continuity should usually need no
card. Prefer direct need or situational resemblance over shared nouns, and do
not select background merely to make the character perform their setting.

Select at most one card. Return only the required JSON.
